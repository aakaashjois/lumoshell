# CLI Reference

```sh
lumoshell <command> [options]
```

## Commands

- `lumoshell apply [--dry-run] [--verbose]`
- `lumoshell setup [--reset]`
- `lumoshell status`
- `lumoshell logs` (prints sync-agent logs from the last 1 hour)
- `lumoshell uninstall`
- `lumoshell doctor`
- `lumoshell version`
- `lumoshell help`

`lumoshell`, `lumoshell help`, and `lumoshell --help` print the usage summary.

## Helper Binaries
